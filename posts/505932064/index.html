pandas

1、nan != nan，所以 query 中这个语句可以筛选空值。

2、pandas 的均值分母是忽略缺失值的。如果一列全是缺失值，中位数和平均值都是NaN

3、遍历行的办法：

numpy

  1. 一般的切片 x[:3] 产生 view，view 的修改会作用在源数组上。但是 id(view) 和 id(orig) 是不一样的。

  2. numpy 的传参是引用，可以在函数内部对其修改。不过如果在内部重新赋值，那么赋值的结果不会作用到外部变量

os

os.path.expanduser：把 ~ 解析为 /home/user

os.path.abspath：绝对路径

re

re.sub：表达式、替换内容、输入字符串

re.findall：返回所有匹配的字符串列表

re.match：匹配成功返回 True

re.search：返回匹配到的位置下标

字符串处理

str.replace：替换

str.partiton：提供一个字符串，匹配到就进行一次 split（split 会寻找全部，partition 找到一个就停止）

str.strip：剪掉两侧的，默认空格和换行符

str.lower：小写。大写 upper

str.split：有一个 num 参数，默认 = -1 表示分割全部间隔符，也可以设置 = 1 表示从开头开始最多进行一次分割。举例：“1,2,3”.split(’,’, 1) == [‘1’, ‘2,3’]

输入输出

input：获取一行的输入字符串，不包含换行符

sys.stdin

类

1、父类的变量更新会影响到（1）子类（2）实例化的子类。但是实例化的子类一旦重写了对象，那父类的改动就不会在上面生效了

（子类继承来的对象和父类的对象的内存地址是相同的）。子类内部更新对象不会影响父类

2、super的目的是，子类实例化的时候，执行父类的初始化

3、dir(类)，查看全部成员

4、__slots__ ：限定类里面允许实例添加的成员名称。除非在子类中也定义__slots__，这样，子类实例允许定义的属性就是自身的__slots__加上父类的__slots__。

5、

@staticmethod：静态方法，不需要实例化，不会读取第一个参数作为self（不需要self）看成类命名空间下面的函数即可

@classmethod：类方法，不需要实例化，读取第一个参数作为cls本身，可以调用cls的成员

@property：get 方法，需要实例化，不需要括号调用方法，返回值

itertools

chain：可变参数数目，串联多个迭代器，按顺序输出他们

product：笛卡尔积，返回 m x n 种的一一配对

collections

set

哈希表，set 不支持索引，没办法取元素。适合查找。

set.intersection：交集

set.union：并集

set.add / set.remove：加入、删除元素

内置函数

all

判断可迭代对象内所有元素是不是都是 True。类似 np.all

zip

输入多个可迭代对象，把他们的元素打包成 tuple。新迭代器的长度取决于输入最短的那个

List

index： 类似 find，查找元素所在第一个位置

yield 和 yield from

带有 yield 的函数变成了一个 generator。里面的返回值变成惰性求解，只有在迭代的时候才会执行语句

yield from

用于展开嵌套生成器，如果用生成器生成生成器，会是这样的

第一次展开返回的是迭代器，第二次返回的是对迭代器执行迭代的结果（如果 gen1 再调用别的迭代器，还是要用 from 来展开）

yield from 最大的用途就是展开嵌套数组

其他

  1. 局部（函数里面）访问全局变量，可以。局部修改已经定义的全局变量，不可以
  2. tuple里面不允许改变的是内存位置，但是可以append列表
  3. zip 逐元素打包元组，zip(*zip)解压缩还原
  4. sort：key，可以设置 lambda x: (x[0], -x[1])。第一位升序，第二位降序。tuple、list 把内容裹起来即可
  5. 倒序输出：（1）reverse（2）[::-1]
  6. ord(字符串) 可以将一个基本字符映射到对应的整数；逆映射为：chr(数字/十六进制)，97-122为英文字母
  7. x >> 1 和 x // 2 是等价的
