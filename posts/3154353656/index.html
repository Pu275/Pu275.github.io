一个仿照sklearn的SGDClassifier

发表于 2022-03-17 更新于 2022-03-30 分类于 机器学习 Valine：

SGDClassifier

官方链接：https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.SGDClassifier.html

参考链接：https://leon.bottou.org/projects/sgd#stochastic_gradient_svm

看一下 sklearn.linear_model.SGDClassifier 有哪些功能

梯度下降计算用 Cython 写，其他都是纯 Python。L1 功能的实现是 truncated gradient algorithm，https://aclanthology.org/P09-1054.pdf

参数说明

max_iter： 迭代 epoch 次数的上限，一次 epoch 会对全部样本都进行一次梯度下降。默认值 1000，实际上十几次就收敛了（大概）

shuffle： 每个 epoch 结束后是否打乱样本，随机梯度的随机的来源

partial_fit： 只用一次 epoch 来 fit

GridSearchCV

根据 params 字典生成参数组合（网格），进行交叉验证选出 best_estimator_

先生成训练集的 k 折子样本，

每个网格的格点是一个候选参数字典，传给 set_params 来设置模型

参数说明

refit： 挑完最优模型是否将该参数应用在全模型上

cv： 几折交叉验证

n_jobs： 并行进程数，joblib.Parallel 用到

joblib.Parallel： 并行加速 for 循环，返回一个所有 for 结果的 list

SequentialDataset 类

位于 sklearn.utils._seq_dataset.pyx。

sklearn.linear_model._base 的 make_datasets 根据输入数值类型创建需要的 dataset 类。在 fit_binary 阶段，输入的X 和 y 被做成了 dataset

dataset 提供的功能：

  1. 任意一个样本的取样 _sample
  2. shuffle 打乱数组索引，每个 epoch 结束都要调用此方法
  3. 样本索引数组 index
  4. next 方法取下一个索引的样本，如果完成一个 epoch 则从头开始再取
  5. 样本权重数组 sample_weight

参数说明

index_data_ptr： 记录 shuffle 以后样本的顺序的数组，存放下标

x_data_ptr： 指向下个样本 xix_ixi​ 的指针

x_ind_ptr： 指向下个样本非零特征下标的指针，和 nnz 配合使用（为 0 的特征没必要计算了）

nnz： 下个样本含有的非 0 特征的个数

y： 指向下个样本 yiy_iyi​ 的指针

损失函数类

几个分类器的损失函数在 sklearn.linear_model._sgd_fast.pyx 下面定义，还有一阶导的函数

损失函数都可以表示成一个二元函数：

loss=f(p,y)=f(wTx,y)p=wTxlogistic=log⁡(1+exp⁡(−p⋅y))=log⁡(1+exp⁡(−wTx⋅y))svm=max⁡(0,1−p⋅y)\begin{aligned} & loss=f(p, y)=f(w^T x, y) \\ & p = w^T x \\ & logistic=\log(1 + \exp{(-p\cdot y)})=\log(1+\exp{(-w^Tx\cdot y)}) \\ & svm = \max{(0, 1-p\cdot y)} \end{aligned}​loss=f(p,y)=f(wTx,y)p=wTxlogistic=log(1+exp(−p⋅y))=log(1+exp(−wTx⋅y))svm=max(0,1−p⋅y)​
