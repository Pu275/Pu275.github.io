首尾

扩容

vector 本身也有自动扩容（数组大小）的机制。capacity 查看当前数组大小，size 查看元素个数。可以看到 vector 的长度突破 capacity 的时候会发生扩容，capacity 变为两倍。

访问下标

注意： 方括号可以超出当前元素个数访问不报错！类似于原生数组用指针访问，即使超界仍然可以访问内存，只不过都是未定义的数据，很危险。

即使 vector 已经 clear，内存里的数据还保留着，如果 clear 前用指针记录了 vector 的数据位置，便仍可以访问到。

  2. 拷贝 vector 功能
  3. vector 可以作为任何对象的容器，push_back 新元素调用对象的拷贝构造函数并放进容器

扩容

当 Vector 的元素位置到达 end 时候，触发扩容，开辟一块新的内存，并把旧的 first 看管的内存拷贝过去，释放旧的内存，最后把 first 的位置也移到新内存。
