19. 删除链表的倒数第 N 个结点

发表于 2022-04-23 更新于 2022-04-25 分类于 LeetCode Valine：

题目

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

链接：https://leetcode-cn.com/problems/remove-nth-node-from-end-of-list/

解答

两个概念：

指针指向的值： 指针解引用后的值，指针变量对它有修改权限

指针值： 指针指向的地址，局部可以通过赋值改变指针变量的指向。

二级指针

二级指针有对指针值修改的权限，进行解引用后作为 ListNode 指针移动遍历链表。

ListNode： 一个对象，它的成员有一个 next 指针变量

ListNode*： 一个值，对指向的 ListNode 对象有修改权限

ListNode**： 一个值，对指向的 ListNode 指针有修改权限，对二次指向的 ListNode 对象也有修改权限

可以不用写 dummy 指针，也不用穿针引线了。链表通过结构体里面的 ListNode 指针变量描述结点之间的关系。遍历的时候只用一个 ListNode 指针去顺着 next 的关系移动，但是指针只能修改指向的变量，不能修改其他指针变量（也就无法改变链表结构）。除非用第二个辅助指针调整第一个指针的指向。
